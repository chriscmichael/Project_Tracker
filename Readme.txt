This project was done as a solo project as the module 1 final project for the Hack Reactor
19 week JavaScript and Python bootcamp. This is a website to track your tasks. It is run on
a Django framework with Python on the backend and HTML on the frontend. This was done as a
solo project.

To run in your browser follow these steps:

1. Open the file with your IDE.
2. Create virtual environment.
   Run the command "python -m venv .venv" in your terminal.
3. Activate virtual environment.
   Run the command "source ./.venv/bin/activate"
4. Install dependencies
   Run "pip install -r requirements.txt"
5. Start development server
   Run "python manage.py runserver"
6. Signup
   Type or copy "http://127.0.0.1:8000/accounts/signup/" into your browser.
   You will be redirected to the home page upon successful signup.
